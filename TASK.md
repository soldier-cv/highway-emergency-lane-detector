# Bug修复任务

## 用户反馈的问题

1. **HTML报告中"查看截图"链接点击后图片不存在** — 报告生成后，通过Web界面查看时截图图片404
2. **HTML报告中"举报视频片段"链接点击后视频不存在** — 同上，视频片段链接也404
3. **结果文件列表展示内容不对** — Web界面的结果文件列表显示的文件内容/结构有问题
4. **远程访问时应支持下载全部数据** — 用户通过远程浏览器访问Web服务时，需要能把所有结果数据下载回来

## 运行环境说明

本项目有两条运行路径：
- **本地GUI模式**: `traffic_violation_gui.py` + `run_gpu_v8.py`，用户在本地打开生成的HTML报告
- **Web服务模式**: `web_server.py` + `web/templates/index.html`，用户通过Flask服务远程访问

请先阅读上述相关代码，分析每个问题的根因，然后修复。
